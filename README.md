# 刘思秋的代码仓库

#### 介绍
这个仓库用来放自己在用本征微观态方法研究降雨的相关代码和结果

#### 软件架构
python 3.11 

#### 使用说明

1.  只包含代码和结果,不包含原始的数据以.mat格保存式(matlab 文件版本v7.3之后是以hdf5格式保存的),如果需要可以联系作者

2.  文件pacPicture里面保存了运行的结果,是对2013到2022年的0.25*0.25地球经纬度格点上的三天平均降雨和水蒸气数据,进行pca分解得到的图像(取前20个主成分).
    经度度范围[-180,180],纬度范围[-20,20],分成了四个地区(western Pacific, eastern Pacific, Atlantic Ocean, Indian ocean)
    每个地区分别进行pca分解,得到的图像分别保存在四个以地区命名的文件夹里

3.  作者目前也在探索中,欢迎大家提出意见和建议,代码和结果仅供参考
4.  output.png是四个地区前20个奇异值大小比较
5.  注意,由于卫星数据只有含海洋地区的数据,陆地地区的数据置为0
